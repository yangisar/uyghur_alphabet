🎮 About This Game

An interactive educational game designed to help you learn 5 distinct writing systems that feature the vowels A, E, I, O, U: Uyghur Perso-Arabic script, Old Turkic runes, Devanagari, Georgian Mkhedruli, and Katakana. Master these beautiful alphabets through memory matching games and tracing exercises.

📚 Writing Systems Included

  • Uyghur Kona Yëziq - The traditional Perso-Arabic script used by Uyghur people in Central Asia.
  • Old Turkic (Orkhon) Script - Ancient runic alphabet used by Göktürk, Uyghur, and other Turkic peoples.
  • Devanagari - Widely used script in India and Nepal for languages like Hindi and Sanskrit.
  • Georgian Mkhedruli - The elegant modern script of the Georgian language.
  • Katakana - A Japanese syllabary used primarily for loanwords and emphasis.

🎯 Game Modes

  • Memory Match - Flip cards to match letters in native script with their equivalents. Test your recognition and memory skills.
  • Trace Draw - Practice writing letters by tracing over ghost previews. Each letter gets a unique brush color. Improve your calligraphy and muscle memory.

⚙️ Features

  • Progress Saving - Your level and lives are automatically saved in your browser
  • Stakes Mode - Enable 7 lives for challenge mode. Gain lives by matching 2 pairs in a row. Lose lives only after viewing all cards.
  • Alphabet Reference - Quick reference table showing all letter equivalents across scripts
  • Responsive Design - Works on desktop, tablet, and mobile devices
  • Multiple Content Types - Practice single letters or letter combinations

💡 Tips for Learning

  • Start with single letters before moving to combinations
  • Use the reference sheet frequently at first
  • Practice writing in Trace Draw mode to build muscle memory
  • Enable Stakes mode once you're comfortable for extra challenge
  • Regular short practice sessions are more effective than long cramming

👨‍💻 Technical Details

Built with pure HTML5, CSS3, and JavaScript. No external dependencies except fonts. Uses localStorage for progress persistence. Fully responsive design with touch support for mobile devices.
